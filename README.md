## ローカルでのテスト手順
### サイトの表示
```bash
cd path/to/ThomBrowne
node server.js
```
http://localhost:3000 で確認

### ブラウザでのデバッグ:
ブラウザの開発者ツール（F12）を開き、コンソールタブでlocalStorage.debug = 'app'を実行します。これで、debugライブラリのログがコンソールに表示されるようになります。
