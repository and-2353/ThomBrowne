## ローカルでのテスト手順
### サイトの表示
```bash
cd path/to/ThomBrowne
node server.js
```
http://localhost:3000 で確認

### ブラウザでのデバッグ(ログの見かた)
1. ローカルでサーバー立ち上げ
2. http://localhost:3000にアクセスし、ブラウザの開発者ツール（F12）を開く
3. Consoleタブにログが表示される
